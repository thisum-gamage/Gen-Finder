# 🎂 Age & Generation Finder

A neat and professional Python script that calculates your current age, identifies your generation, and predicts the year you will turn 100.

## 🚀 Key Features
- **Generation Logic:** Accurately maps years to generations (Gen Alpha to Greatest Generation).
- **Age Calculator:** Uses the year 2026 as the base for age calculation.
- **Milestone Tracker:** Predicts your 100th birthday year.
- **Robust Input Handling:** Includes `try-except` blocks to handle non-numeric inputs gracefully.

## 🛠️ Concepts Used
- Modular Functions (`def`)
- Error Handling (`ValueError`)
- f-string Formatting
- Conditional Logic (`if-elif-else`)

## 📖 How to Run
Run the script using Python:
```bash
python Gen.py
